C# Quiz

What is this?

This is an 'underhanded' quiz on the C# language. It focuses on strange edge cases and quirks of the language. Don't feel bad if you aren't able to answer most of these - some of the answers are pretty obscure!

The idea and parts of the code were taken from David Tolnay's excellent Rust Quiz - go check it out.

The source code for this quiz is available on GitHub.
